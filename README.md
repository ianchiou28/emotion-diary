### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/ianchiou28/emotion-diary.git
   cd emotion-diary
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
